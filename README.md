# AI Chatbot for improving resume
(based on the initial project generated by `create-cloudflare`)

An AI-powered chatbot that helps users improve their resumes and CVs. Upload your resume, provide job listings, and get tailored suggestions to better match job requirements.

## Tech Stack
- **Frontend**: React, Vite, TypeScript, Tailwind CSS
- **Backend**: Cloudflare Workers, Durable Objects (SQLite)
- **AI Model**: Google Gemini (gemini-3-flash-preview)

## Local Testing
1. Create `.dev.vars` in the project root folder and set `GOOGLE_GENERATIVE_AI_API_KEY` to be the gemini key.
2. Run `npm install` in the project root to install dependencies
3. Run `npm start` to start local server

## Project Structure

```
├── src/
│   ├── app.tsx        # Chat UI implementation
│   ├── server.ts      # Chat agent logic
│   ├── tools.ts       # Tool definitions
│   ├── utils.ts       # Helper functions
│   └── styles.css     # UI styling
├── wrangler.jsonc     # Cloudflare Workers configuration
└── .dev.vars          # Local environment variables (not committed)
```

## Limits
Maximum file upload size: 2MB (Durable Objects SQLite string limit)
